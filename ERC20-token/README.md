# TPN

## Initial Demo

### Initial Development

```
$ npm install
```

Complile Solidity Contract (TPNDemoToken)

```
$ npm run compile
```

Deploy Demo Contract to Local Ganache:

```
$ npm run migrate-ganache
```

Deploy Demo Contract to Ropsten Testnet:

```
$ npm run migrate-ropsten
```
